## Note: hephaes Phase 6 (Image Draft Spec Fixes)

Three hephaes library bugs were fixed in the library layer and flow through automatically. One backend fix was also required:

- [x] Fix `_open_asset_reader` context manager exception handling in `conversion_authoring.py`: the generic `except Exception` block around the `yield` was catching re-raised `ConversionAuthoringServiceError` subclasses (e.g., validation errors from draft generation), wrapping them as misleading `"failed to open asset reader"` messages. Fixed by letting all `ConversionAuthoringServiceError` subclasses propagate.
